Open T-Cards

Open T-Cards is a digital T-Cards implementation using Swing components and standard serialization techniques to save and sort data used for task management.

The following are some of the features that I plan on implementing. The list is not conclusive, though it is fairly complete. I will try to keep it updated as feature requests are accepted. Though as the project is in its incubation stage currently, I do not suspect that the list will change much in the near future.

  • XML-based project files
  • Project collections
  • Single boards per project
  • Multiple columns per board
  • Unlimited cards per column
  • Editable column labels
  • Dynamic attribute lists for each card
  • Ability to pre-set attribute lists for new cards

The GUI is laid out using Swing and a combination of box and border layouts for layout management. This project is fairly small at the moment, and so I don't really feel like I need much help. Nonetheless, if you feel like you can add to the project, please feel free to request a role, and let me know what your thoughts are. Thank you.
