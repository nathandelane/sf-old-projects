Browser Information

In the world of web design, it has been a tradition to be difficult to provide a similar user interface across multiple browsers, especially of different brands. However with the advent of web standards, it makes sense to follow those, as the web continues forward. As such I make no guarantee that my work will work exactly the same on old browsers. Old browsers include, but are not limited to Internet Explore 6.0 and older, Firefox 1.5 and older, Opera 8 and older, and Netscape Navigator (any version).

These browsers do not support web standards, and so I can make no guarantee that my work will function or look the same in these or similar browsers. As such I advise you and all of your patrons and customers to upgrade to a browser that does support web standards. This is part of a global movement to standardize the web and provide accessibility features that everyone can be happy about. To learn more about the web standards movement, please follow this link to The Web Standards Project.

Thanks.

Nathan Lane
